# todo-List

A simple Electron-based Todo List application.

## Requirements

- Node.js >= 18
- npm 

### Install node.js

Install Node.js (>= 18) from: https://nodejs.org/
Fedora: sudo dnf install nodejs 

### Setup

1. Clone the repo
   git clone https://github.com/xinran0606/todo-List.git

2. Install dependencies
   npm install

3. Start the app
   npm start
